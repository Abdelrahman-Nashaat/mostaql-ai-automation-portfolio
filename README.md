# Mostaql AI Automation Portfolio

Public demo hub for Mostaql portfolio works.

The goal is to give clients visible evidence instead of plain screenshots:

- Live static demos for each portfolio item.
- Source code that can be reviewed.
- Clear result summaries for every demo without claiming unverified production metrics.

## Demo Pages

- `ai-leads-dashboard.html` - AI lead qualification chatbot and Airtable/Google Sheets handoff.
- `foodics-analytics-api.html` - Foodics API, PostgreSQL/Supabase, multi-tenant analytics middleware.
- `watch-store-admin.html` - watch store MVP with admin view and WhatsApp ordering concept.
- `n8n-jobs-wordpress.html` - n8n workflow publishing job news to WordPress.
- `whatsapp-sheets-assistant.html` - WhatsApp assistant that answers from Google Sheets.
- `invoice-ocr-workbench.html` - invoice OCR review workflow.
- `crm-lead-agent.html` - AI agent for lead classification and CRM updates.

## Notes

These demos are static front-end proof-of-work samples. They are intentionally transparent: they show UI, workflow, expected outputs, and code structure without pretending to be full production deployments.
