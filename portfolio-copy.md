## AI lead qualification bot

Title:
بوت ذكاء اصطناعي لتأهيل العملاء وربطه مع Airtable أو Google Sheets

Description:
نموذج عملي لنظام أتمتة يجمع بيانات العملاء من المحادثات، يطرح أسئلة تأهيل، يحسب درجة أولوية لكل عميل، ثم يرسل البيانات المنظمة إلى Airtable أو Google Sheets.

يشمل العمل:
1. تصميم تدفق المحادثة.
2. منطق تصنيف العميل حسب الميزانية والاحتياج والموعد المتوقع.
3. Webhook لاستقبال الرسائل وربطها بلوحة متابعة.
4. تصدير البيانات إلى CRM أو جدول متابعة.
5. لوحة بسيطة لعرض العملاء المؤهلين وحالة المتابعة.

Skills:
الذكاء الاصطناعي, الأتمتة, واجهة برمجة التطبيقات, Google Sheets, Airtable, تصميم موقع إلكتروني

Skills:
واجهة برمجة التطبيقات, PostgreSQL, Supabase, برمجة مواقع, تطوير البرمجيات, هندسة البرمجيات

## n8n jobs to WordPress automation

Title:
أتمتة n8n لجلب أخبار الوظائف ونشرها في ووردبريس

Description:
نموذج عملي لتدفق n8n يجلب أخبار الوظائف من مصادر متعددة، ينظف البيانات، يمنع التكرار، يستخدم الذكاء الاصطناعي للتصنيف والوسوم، ثم ينشر المحتوى في ووردبريس عبر REST API.

يشمل العمل:
1. جلب البيانات من RSS أو API أو صفحات HTML.
2. تنظيف العنوان والمحتوى واسم الشركة والصورة.

4. منع تكرار النشر باستخدام رابط الخبر أو hash للمحتوى.
5. نشر المقالات في ووردبريس مع سجل للأخطاء.

Skills:
n8n, WordPress, واجهة برمجة التطبيقات, الذكاء الاصطناعي, الأتمتة, برمجة مواقع
